# 🐼 เกมเรียนคำศัพท์ภาษาจีน (N1 เทอม 3)

เว็บแอปฝึกคำศัพท์ภาษาจีนสำหรับเด็ก 3 ขวบ เตรียมสอบ — รูปภาพดึงมาจากไฟล์ PowerPoint บทเรียน (สัปดาห์ 2–7)

## วิธีเล่น
เปิดไฟล์ **`index.html`** ด้วยเบราว์เซอร์ (ดับเบิลคลิกได้เลย) — แนะนำ Google Chrome เพื่อให้มีเสียงอ่านภาษาจีน

## คำศัพท์ 12 คำ · 3 หมวด
| หมวด | คำศัพท์ |
|---|---|
| 🌤️ ธรรมชาติ (大自然) | 星星 ดาว · 月亮 พระจันทร์ · 太阳 พระอาทิตย์ · 白云 เมฆ |
| 🌳 พืช (植物) | 大树 ต้นไม้ · 爱 รัก · 红花 ดอกไม้ · 绿草 หญ้า |
| 🚗 ยานพาหนะ (交通) | 飞机 เครื่องบิน · 小船 เรือ · 汽车 รถยนต์ · 巴士 รถบัส |

## โหมดในแอป
1. **📖 ทบทวนคำศัพท์** — ดูรูป + อ่านตัวจีน/พินอิน/คำแปล + ฟังเสียงอ่าน (ทบทวนก่อนเล่นเกม)
2. **🔊 ฟังเสียง เลือกรูป** — เกมฝึกฟัง
3. **🖼️ ดูรูป เลือกคำ** — เกมฝึกอ่านตัวอักษร

มีดาวสะสม 🌟 เสียงชม และเอฟเฟกต์ฉลองเมื่อตอบถูก · ตอบผิดไม่หักคะแนน ให้ลองใหม่ได้เรื่อยๆ

## ไฟล์ที่จำเป็นต่อการใช้งาน
`index.html` · `styles.css` · `app.js` · `data.js` · `assets/img/*.png`
